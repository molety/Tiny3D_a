# Tiny3D_a
Tiny 3D demo for Java Applet

## Operation

- ↑: Look up
- ↓: Look down
- ←: Look left
- →: Look right
- H: Tilt to the left
- L: Tilt to the right
- K: Zoom in
- J: Zoom out
- 0: Return to the initial position

- 1: 1 screen mode
- 2: 2 screens mode

- space: operation manual
- Q: quit
